Пояснительная записка.
Main - основной цикл программы. С его помощью отображаются все возможные окна. in_menu - переключатель. Его значение меняется по нажатию кнопок.
Теперь по значениям in_menu и окнам.
При in_menu ==  1 открывается главное меню программы. Из него можно перейти во все остальные окна игры. В поле для ввода вписывается ник игрока, который в последствии будет использован для отображения в таблице лидеров. Два последующих drop down menu отвечают за выбор уровня и скина для динозавра. По нажатию кнопки Играть! запускается уровень, а in_menu ==  0. Из txt файлов идет подгрузка уроыней для игры по выбору пользователя. По завершении прохождения уровня ( достижения зеленой клетки ) появляется кнопка для возврата в главное меню.
При нажатии на кнопку Правила открывается окно с краткими правилами игры. Оттуда также можно вернуться в главное меню. 
При нажатии на кнопки Топ первого уровня и Топ второго уровня открывается окно со списком лидеров по наименьшему количеству прыжков на данных уровнях. Данные подгружаются из базы данных с помощью sqlite3. Из них можно вернуться в главное меню с помощью кнопки back2.
